📖 Como usar o Importador BotConversa

📂 1. Importar arquivo

Arraste um arquivo .xlsx, .xls ou .csv para a área de upload, ou clique para selecionar. O arquivo é lido diretamente no seu navegador — nenhum dado é enviado para a internet.

🗂 2. Mapear colunas

  • Selecione qual coluna do arquivo corresponde a Nome, Telefone e (opcionalmente) Etiquetas.
  • Nome e Telefone são obrigatórios — sem eles não é possível prosseguir.
  • Etiquetas é opcional. Mesmo sem mapear, a coluna estará disponível na etapa 3 para preenchimento manual.
  • Use o botão "Limpar Seleção" para reiniciar todos os campos e começar do zero.

✏ 3. Editar etiquetas

  • Clique no + ao lado das etiquetas para adicionar uma nova.
  • Escolha aplicar somente na linha ou em todas as linhas.
  • Clique no × de uma etiqueta para removê-la.
  • Duplo clique em uma etiqueta existente para renomeá-la.
  • Pressione Delete em qualquer célula de etiqueta para limpar todas as etiquetas daquela linha.
  • Use a barra de busca para filtrar registros por nome ou etiqueta.

📤 4. Exportar

  • O arquivo é gerado no formato BotConversa: Primeiro Nome · Sobrenome · Telefone · Etiquetas.
  • Telefones múltiplos separados por / geram linhas individuais.
  • Etiquetas múltiplas ficam na mesma célula separadas por , (vírgula + espaço).
  • Baixe como .xlsx ou .csv.

🗑 Gerenciar Etiquetas Globais

  • Clique em "Gerenciar Etiquetas" na toolbar da etapa 3 para abrir o painel.
  • Remover etiqueta específica: selecione pelo dropdown (lista todas as etiquetas em uso) ou digite o nome manualmente e clique em "Remover de todos" — a etiqueta é removida de cada linha que a continha.
  • Limpar tudo: apaga todas as etiquetas de todos os registros de uma vez. Uma confirmação é solicitada antes de executar.
